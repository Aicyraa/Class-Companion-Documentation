About

Class Companion is a Discord bot designed to help students manage their class schedules, reminders, and activities in one place. It assists by organizing daily routines, sending timely notifications, and making class coordination much easier for both students and teachers.

The bot uses a simple // prefix before commands. For example, when you type //schedule, the bot replies with a list of available commands:

Schedule

This section allows you to manage your class schedules. You can view your timetable, update existing schedules, or add new activities using the provided commands. It ensures that you never miss an important class or event.

View Schedule

Displays the list of all upcoming classes, activities, or events you have set. This feature helps students quickly check their daily or weekly timetable.
